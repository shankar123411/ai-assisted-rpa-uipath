# AI-Assisted Robotic Process Automation using UiPath

## Overview
End-to-end RPA solution built with UiPath Studio that eliminates 
repetitive manual data processing tasks using AI-based text 
classification.

## Key Features
- Automated Excel data processing and report generation
- Reduced manual effort by 70%
- AI-based text classification module for smarter data processing
- Dynamic conditional branching for diverse data scenarios
- Complete solution: requirements → development → testing → deployment

## Technologies Used
- UiPath Studio
- Python (OOP)
- MS Excel
- AI Text Classification

## System Architecture
Multi-agent design:
- Orchestrator Agent — routes and coordinates tasks
- RPA Bot Agent — executes UiPath workflows
- AI Classifier Agent — text and data analysis
- Data Agent — store and retrieve data
- Validation Agent — quality and accuracy check

## Results
- 70% reduction in manual effort
- Automated end-to-end report generation pipeline
